## Detailed Cost Breakdown

### 1. Hosting Costs

#### WordPress Hosting Options

**Option A: Shared Hosting (Bluehost, SiteGround, HostGator)**
- Cost: $2.99-30/month
- Average: **$100-300/year**
- Pros: Cheap, easy setup, WordPress-optimized
- Cons: Limited resources, slow performance, security concerns

**Option B: Managed WordPress Hosting (WP Engine, Kinsta, Pressable)**
- Cost: $20-300/month
- Average: **$240-3,600/year**
- Pros: Optimized performance, better support, automatic updates
- Cons: Expensive, overkill for many sites, vendor lock-in

**Option C: VPS or Cloud (AWS, Linode, DigitalOcean)**
- Cost: $5-50+/month
- Average: **$100-600/year**
- Pros: Full control, scalable, good performance
- Cons: Requires technical knowledge, you manage everything

**WordPress hosting average:** $240-600/year

#### Hugo Hosting Options

**Option A: GitHub Pages**
- Cost: $0/month
- Annual: **$0**
- Pros: Free, unlimited bandwidth, automatically scales
- Cons: GitHub only, no custom back-end

**Option B: Netlify Free Tier**
- Cost: $0-20/month
- Annual: **$0-240**
- Pros: Free tier works for most sites, excellent Hugo support
- Cons: Bandwidth limits at scale, paid tier needed for large sites

**Option C: Vercel**
- Cost: $0-20+/month
- Annual: **$0-240**
- Pros: Excellent performance, edge functions
- Cons: Paid plans needed for complex features

**Option D: CloudFront CDN (for premium performance)**
- Cost: $0.085 per GB
- Annual (100GB/month traffic): **~$100**
- Pros: Global distribution, enterprise-grade
- Cons: Only needed for high-traffic sites

**Hugo hosting average:** $0-100/year (or $0-600 with premium CDN)

**Hosting advantage: Hugo saves $240-600/year**

- Update plugins (and test): 2 hrs
- Check theme updates: 0.5 hrs

- Monitor security: 1 hr

- Check for content updates needed: 0.5 hrs
- Deploy new content: 0.5 hrs
- Monitor performance: 0.25 hrs
- Review analytics: 0.25 hrs

**Total: 1-2 hours per month = 12-24 hours per year**

**Salary cost (assume $100/hour):**
$1,200-2,400 per year

### 4. Security, Backup & Disaster Recovery

#### WordPress Costs

**Backup services:**
- Manual backups: ~5 hrs/month = $600/year
- Backup plugins: $50-200/year
- Backup storage: $0-100/year

**Security monitoring:**
- Security plugin: $99-300/year
- Security audits: $0-1,000/year

**Incident recovery (if hacked):**
- Professional recovery: $500-2,000
- Downtime/lost sales: $1,000-10,000+
- Reputational damage: Varies

**Probability of being hacked: 15-30% over 5 years**

**Expected cost: $75-500/year (averaged over time)**

**WordPress security cost: $200-2,000/year (+ potential incident costs)**

#### Hugo Costs

**Backup services:**
- Git provides version control: $0
- Automated backups: $0 (GitHub/Netlify handle this)

**Security monitoring:**
- No vulnerabilities to monitor: $0

**Incident recovery:**
- Much lower likelihood (static files)
- If compromised, revert to previous version: $0

**Probability of incident: < 1%**

**Hugo security cost: $0-100/year**

**Security advantage: Hugo saves $200-1,900/year**

### 5. Development/Customization Costs

#### WordPress Customization

If you need custom functionality:
- Plugin development: $500-5,000+
- Custom theme development: $1,000-10,000+
- API integrations: $500-5,000+

WordPress is extensible but often requires expensive custom development.

#### Hugo Customization

Hugo is highly customizable with templating:
- Custom layouts: $500-2,000
- Custom features: $500-5,000+
- API integrations: $500-5,000+

Hugo customization is often cheaper because you control everything without plugin layers.

**Note:** For simple sites, customization costs are similar. For complex sites, Hugo often wins.

## Sensitivity Analysis: What Matters Most?

### What if you already have a WordPress site?

The migration cost is real, but typically recovers quickly:
- Small site migration: $1,500-3,500
- Medium site migration: $5,000-10,000
- Large site migration: $12,000-25,000+

**Break-even analysis:**
- Small business: Savings cover migration cost in ~3 months
- Medium business: Savings cover migration cost in ~3-6 months
- Large organization: Savings cover migration cost in ~6-12 months

After break-even, you're saving thousands per year.

### What if you only care about hosting costs?

Even if you ignore maintenance time, Hugo saves:
- **$240-600/year** on hosting alone
- **Over 5 years: $1,200-3,000**

This alone justifies migration for performance-conscious organizations.

### What if you value your time at $200/hour?

WordPress maintenance time becomes expensive:
- **$10,800-21,600/year** just in maintenance labor
- With this calculation, Hugo saves **$8,000-20,000/year**

## Hidden Costs We Haven't Discussed

### WordPress Hidden Costs:

1. **Performance issues** - Slow sites lose customers (conversion impact: $1,000s)
2. **Downtime** - WordPress sites can go down (lost sales, brand damage)
3. **Security breaches** - Average cost: $4.29 million (IBM, 2023)
4. **Compliance** - GDPR/HIPAA compliance is harder with WordPress
5. **Scalability** - As traffic grows, WordPress becomes expensive to scale
6. **Lock-in** - Migrating off WordPress to something else is costly

### Hugo Advantages:

1. **Better SEO** - Fast sites rank better (more organic traffic)
2. **Higher conversions** - Fast sites convert better (3-5% improvement typical)
3. **No breach risk** - Static sites eliminate entire categories of risk
4. **Easy export** - Content is plain text, never locked in
5. **Scales for free** - Static files scale infinitely without cost
6. **Portability** - Can switch hosting with zero work
